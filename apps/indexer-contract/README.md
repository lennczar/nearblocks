## Indexer Contract

Contract indexer collects the info about NEAR contract state changes

### Config

```
DATABASE_URL=
NETWORK=mainnet
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=

# Optional
DATABASE_CA=
DATABASE_CERT=
DATABASE_KEY=
CONTRACT_START_BLOCK=
CONTRACT_DATA_SOURCE=   # NEAR_LAKE | FAST_NEAR
SENTRY_DSN=
```
